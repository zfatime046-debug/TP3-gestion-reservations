# Système de Réservation de Salles - Hibernate

## Description
Ce projet est une application Java utilisant **Hibernate ORM** pour gérer un système de réservation de salles.  
Il démontre l'utilisation des relations entre entités et des opérations en cascade.

## Technologies utilisées
- Java
- Hibernate
- JPA
- H2 Database
- Maven

## Modèle de données

### Entités principales
- **Utilisateur**
- **Salle**
- **Reservation**
- **Equipement**

### Relations
- Un **Utilisateur** peut avoir plusieurs **Réservations** (OneToMany)
- Une **Salle** peut avoir plusieurs **Réservations** (OneToMany)
- Une **Salle** peut avoir plusieurs **Équipements** (ManyToMany)
